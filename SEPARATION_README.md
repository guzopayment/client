# Questionnaire Management Client

This bundle is the questionnaire-only frontend.

## Routes
- `/` questionnaire home
- `/questionnaire` public questionnaire form
- `/thank-you` questionnaire thank you page
- `/admin-login` admin login
- `/admin-questionnaire` questionnaire admin management
- `/questionnaire-view-dashboard` grouped organization dashboard
- `/admin-questionnaire-print` printable questionnaire table
- `/admin-history` questionnaire history log

## Local run
```bash
npm install
cp .env.example .env.local
npm run dev
```

## Build
```bash
npm run build
```
